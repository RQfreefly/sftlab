功能模块

参数管理

保存与编辑 SFT CLI 参数模板，支持版本记录与基础校验。

Prompt 管理

按目录分类维护 Prompt，支持 CRUD 与历史版本管理。

Token 统计

支持 gpt、qwen、llama 三类模型的 Token 计数。

JSON 工具

提供解析、格式化、压缩、转义/反转义与错误提示。

Diff 工具

支持行级/字符级文本对比，并可进行 JSON/Prompt 归一化比较。

LLM API 测试

支持 OpenAI 兼容接口请求与流式响应解析。

Calculator

支持表达式计算、变量管理与历史记录。

Timer

支持分段计时与历史会话记录。

典型使用流程

  1. 配置并保存 SFT 参数模板
  2. 管理 Prompt 并完成版本迭代
  3. 统计 Token，处理 JSON/Diff，并辅助计算与计时
  4. 使用 LLM API 测试验证请求与响应
